# Project250

Ferramentas para implantação e configuração automatizada de sistemas AMD BC250 rodando Bazzite.

## Recursos

- Kernel BC250 customizado
- Desbloqueio de 40 CUs
- Skillfish Tuner
- Telemetria ESP32 + GC9A01
- Configuração automática de serviços
- EmuDeck e emuladores
