# Skillfish Tuner 2.0

Skillfish Tuner is a BC-250 focused tuning utility for Bazzite/Fedora Atomic systems.

## Start

```bash
./INSTALL_OR_RUN.sh run
```

## Install desktop launcher

```bash
./INSTALL_OR_RUN.sh install
```

After installing, launch **Skillfish Tuner** from your application menu or run:

```bash
skillfish-tuner
```

## Included features

- Dashboard with BC-250 system information
- CU manager with factory/last-good restore support
- GPU overclock controls with 700 mV minimum voltage guard
- Fan control with apply/save and optional reboot persistence
- Governor management
- Service health overview
- System report generation
- Fishbone app/tray icon support

## Requirements

- x86_64 / 64-bit Linux
- Bazzite or Fedora Atomic recommended
- Python 3 with GTK/Adwaita bindings
- BC-250 / Cyan Skillfish hardware for full functionality
