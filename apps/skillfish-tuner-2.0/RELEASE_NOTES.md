# Skillfish Tuner 2.0

End-user release package.

Highlights:

- Cleaner dashboard with system information
- Simplified CU status gauge and restore workflow
- Fan settings can be applied, saved, and restored after reboot
- Cleaner Service Health states
- Benchmarking/stability-test tooling removed from the app
- Voltage controls are clamped to 700 mV minimum
- AppRun and install helpers are executable
